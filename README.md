# SQL-Builder-Frontend



## 前端技术

- React 18
- Umi 4.x
- Ant Design 4.x 组件库
- Ant Design Pro Components 高级组件
- TypeScript 类型控制
- Eslint 代码规范控制
- Prettier 美化代码

依赖库：

- monaco-editor 代码编辑器
- copy-to-clipboard 剪切板复制



### 快速启动

- 安装依赖命令：

```bash
npm install
```

- 运行命令：

```bash
npm run dev
```

- 项目地址一般为(在端口不冲突的情况下) :http://localhost:8000/ 

## 环境要求

1. node版本不低于16
2. npm版本不低于8
